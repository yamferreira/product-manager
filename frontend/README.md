# Products Front-end ⚛️

Front-end em **React** para exibir e gerenciar produtos do projeto `Products`. Interface simples e responsiva para consumir a API e realizar operações de CRUD.

---

## 🛠 Tecnologias

![React](https://img.shields.io/badge/React-61DAFB?style=for-the-badge&logo=react&logoColor=black)

---

## 🚀 Funcionalidades

- **Integração com back-end Java** (`products`)  
  - Listagem de produtos  
  - Criação, edição e exclusão de produtos  

> ⚠️ Importante: o back-end deve estar rodando para que o front funcione corretamente.

---
