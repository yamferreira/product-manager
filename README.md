# Products API 🛠️

API backend em **Java (Spring Boot)** para gerenciamento de produtos. Suporta **CRUD completo de produtos**, incluindo criação, edição e exclusão.

---

## 🛠 Tecnologias

![Java](https://img.shields.io/badge/Java-E34F26?style=for-the-badge&logo=java&logoColor=white)
![Spring Boot](https://img.shields.io/badge/SpringBoot-6DB33F?style=for-the-badge&logo=spring&logoColor=white)

---

## 🚀 Funcionalidades

- **CRUD de produtos**  
  - Adicionar novos produtos  
  - Editar produtos existentes  
  - Excluir produtos  
  - Listar produtos disponíveis  

- **Integração com front-end React**  
  - Para que o sistema funcione corretamente, **é necessário iniciar o front-end** esta do lado desse projeto.  

---

## 📝 Instruções para rodar o back-end

1. Abra o projeto na sua IDE (IntelliJ, Eclipse, VScode etc.)  
2. Clique no **método `run` da classe principal** para iniciar a aplicação  
3. Certifique-se de que o **front-end React** esteja rodando em paralelo para que as requisições funcionem
